# Politique de confidentialité de NavXS

Dernière mise à jour : 16 mai 2026

## 1. Informations générales

NavXS est une application Android qui peut afficher un overlay de navigation avec des boutons fixes pour les applications sélectionnées.

Le développeur de cette application ne collecte, stocke ou traite des données personnelles que dans la mesure techniquement nécessaire au fonctionnement de l'application.

## 2. Traitement local sur l'appareil

NavXS traite généralement les informations localement sur l'appareil.

Cela inclut notamment :

- le nom de package de l'application actuellement au premier plan
- la sélection des applications pour lesquelles l'overlay doit être activé
- les paramètres d'overlay définis par l'utilisateur, tels que les positions, couleurs, tailles, transparences et thèmes
- l'état d'activation du service d'accessibilité

## 3. Finalité du traitement

Ces informations sont traitées uniquement pour :

- détecter si l'overlay doit être affiché pour l'application actuellement ouverte
- afficher ou masquer l'overlay
- fournir les actions de navigation configurées
- sauvegarder et restaurer localement les paramètres choisis par l'utilisateur

## 4. Service d'accessibilité

NavXS utilise un service d'accessibilité Android.

Ce service est utilisé exclusivement pour :

- détecter le nom de package de l'application actuellement en premier plan
- décider si l'overlay doit être affiché
- déclencher les actions de navigation prévues

Ce service n'est pas utilisé à des fins publicitaires, de suivi ou d'analyse.

## 5. Aucune transmission à des serveurs externes

Actuellement, NavXS ne transmet pas les applications sélectionnées, les noms de packages des applications en premier plan, ni les paramètres d'overlay à des serveurs externes ou à d'autres services externes.

Le traitement s'effectue localement sur l'appareil.

## 6. Stockage des données

Les paramètres et sélections d'applications stockés par NavXS sont enregistrés localement sur l'appareil.

Ces données restent sur l'appareil jusqu'à ce qu'elles soient modifiées par l'utilisateur, supprimées en effaçant les données de l'application, ou effacées lors de la désinstallation de l'application.

## 7. Autorisations

NavXS requiert notamment l'autorisation d'accessibilité explicitement activée par l'utilisateur afin de fournir sa fonctionnalité d'overlay.

De plus, seuls les accès techniquement nécessaires au fonctionnement de l'application sont utilisés.

## 8. Tiers

Actuellement, NavXS n'utilise aucun service tiers, outil d'analyse ni publicité pour traiter les données décrites ci-dessus.

## 9. Modifications de cette politique de confidentialité

Cette politique de confidentialité peut être mise à jour si nécessaire pour refléter les modifications de l'application ou les exigences légales.
