# NavXS 개인정보 처리방침

최종 업데이트: 2026년 5월 16일

## 1. 일반 정보

NavXS는 선택한 앱에 대해 고정 버튼이 있는 탐색 오버레이를 표시할 수 있는 Android 앱입니다.

이 앱의 개발자는 앱 기능에 기술적으로 필요한 범위 내에서만 개인 데이터를 수집, 저장 또는 처리합니다.

## 2. 기기에서의 로컬 처리

NavXS는 일반적으로 기기에서 로컬로 정보를 처리합니다.

이는 특히 다음을 포함합니다:

- 현재 포그라운드에서 실행 중인 앱의 패키지 이름
- 오버레이를 활성화해야 하는 앱 선택
- 위치, 색상, 크기, 투명도 및 테마와 같은 사용자 정의 오버레이 설정
- 접근성 서비스 활성화 여부 상태

## 3. 처리 목적

이 정보는 다음 목적으로만 처리됩니다:

- 현재 열려 있는 앱에 대해 오버레이를 표시해야 하는지 감지
- 오버레이 표시 또는 숨기기
- 구성된 탐색 작업 제공
- 사용자가 선택한 설정을 로컬에 저장 및 복원

## 4. 접근성 서비스

NavXS는 Android 접근성 서비스를 사용합니다.

이 서비스는 다음 목적으로만 사용됩니다:

- 현재 포그라운드에서 실행 중인 앱의 패키지 이름 감지
- 오버레이 표시 여부 결정
- 의도된 탐색 작업 실행

이 서비스는 광고, 추적 또는 분석 목적으로 사용되지 않습니다.

## 5. 외부 서버로의 전송 없음

현재 NavXS는 선택된 앱, 포그라운드 앱의 패키지 이름 또는 오버레이 설정을 외부 서버나 기타 외부 서비스로 전송하지 않습니다.

처리는 기기에서 로컬로 이루어집니다.

## 6. 데이터 저장

NavXS가 저장하는 설정 및 앱 선택 사항은 기기에 로컬로 저장됩니다.

이 데이터는 사용자가 변경하거나, 앱 데이터 삭제로 제거하거나, 앱 제거로 삭제될 때까지 기기에 남아 있습니다.

## 7. 권한

NavXS는 앱이 오버레이 기능을 제공할 수 있도록 사용자가 명시적으로 활성화한 접근성 권한을 특히 필요로 합니다.

또한 앱 운영에 기술적으로 필요한 액세스만 사용됩니다.

## 8. 제3자

현재 NavXS는 위에 설명된 데이터 처리를 위해 제3자 서비스, 분석 도구 또는 광고를 사용하지 않습니다.

## 9. 이 개인정보 처리방침의 변경

이 개인정보 처리방침은 앱의 변경 사항이나 법적 요구 사항을 반영하기 위해 필요한 경우 업데이트될 수 있습니다.
